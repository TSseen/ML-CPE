# ใบงานที่ 2: Data Preprocessing (การเตรียมข้อมูลสำหรับการทำ Machine Learning)

โปรเจกต์นี้จัดทำขึ้นเพื่อการศึกษาและเรียนรู้เกี่ยวกับกระบวนการทำ **Data Preprocessing** ซึ่งเป็นขั้นตอนที่สำคัญที่สุดขั้นตอนหนึ่งในการเตรียมข้อมูลดิบ (Raw Data) ให้มีความถูกต้อง แม่นยำ และพร้อมใช้งานก่อนนำไปพัฒนารูปแบบจำลอง (Machine Learning Models)

## 🎯 วัตถุประสงค์ของใบงาน (Project Objectives)

1. **เข้าใจหลักการและความสำคัญของ Data Preprocessing:** ศึกษากระบวนการตรวจสอบ ทำความสะอาด แปลงรูปแบบ และเตรียมข้อมูลให้มีคุณภาพถูกต้องตามหลักการก่อนนำไปใช้กับโมเดล Machine Learning (ML)
2. **วิเคราะห์และตรวจสอบคุณภาพของข้อมูลได้:** สามารถตรวจหาจุดบกพร่องในชุดข้อมูล ได้แก่:
   * ข้อมูลสูญหาย (Missing Values)
   * ข้อมูลซ้ำซ้อน (Duplicate Data)
   * ข้อมูลที่ผิดปกติ (Outliers)
   * รูปแบบข้อมูลที่ไม่สอดคล้องกัน (Inconsistent Data) พร้อมทั้งเลือกวิธีแก้ไขได้อย่างเหมาะสม
3. **ประยุกต์ใช้เทคนิคการเตรียมข้อมูลพื้นฐาน:** สามารถเลือกใช้และลงมือทำเทคนิคต่างๆ เพื่อปรับข้อมูลให้อยู่ในรูปแบบที่พร้อมสำหรับการวิเคราะห์ ได้แก่:
   * **Data Cleaning** (การทำความสะอาดข้อมูล)
   * **Data Transformation** (การแปลงรูปข้อมูล)
   * **Data Encoding** (การเข้ารหัสข้อมูลหมวดหมู่ให้เป็นตัวเลข)
   * **Feature Scaling / Data Normalization** (การปรับช่วงสเกลของข้อมูล)
4. **พัฒนาทักษะการโปรแกรมด้วยภาษา Python:** สามารถเขียนโค้ดภายใต้กระบวนการ Data Preprocessing โดยเลือกใช้ไลบรารีที่เกี่ยวข้องได้อย่างถูกต้อง พร้อมทั้งตรวจสอบและประเมินคุณภาพข้อมูลก่อนนำไปใช้งานจริงได้
5. **นำเสนอผลงานผ่านแพลตฟอร์มออนไลน์:** ฝึกฝนการนำเสนอทักษะและโค้ดผ่านแพลตฟอร์มสากล เช่น **GitHub**, **Kaggle** หรือ **Medium** เพื่อจัดเตรียมเอกสารประกอบและสร้างโครงสร้างผลงาน (Portfolio) สำหรับใช้ในการสมัครงานหรือเผยแพร่ผลงานวิชาการในอนาคต

## 🛠️ ขั้นตอนการทดลอง

### 📂 LAB 1: Dataset Exploration (การสำรวจชุดข้อมูล)
* เรียนรู้การโหลดข้อมูลและการตรวจสอบโครงสร้างเบื้องต้นด้วย `.shape` และ `.dtypes`
* วิเคราะห์ค่าสถิติเชิงพรรณนา (Summary Statistics) และตรวจพบคอลัมน์ที่มีข้อมูลสูญหาย (Missing Values)

### 📊 LAB 2: Data Visualization (การแสดงผลข้อมูลด้วยภาพ)
* สร้าง **Histogram** เพื่อวิเคราะห์การกระจายตัวของข้อมูลเชิงปริมาณ
* สร้าง **Correlation Heatmap** เพื่อดูความสัมพันธ์ระหว่างตัวแปรต่างๆ ว่าส่งผลต่อกันอย่างไร

### 🧼 Part 3: Data Cleaning (การทำความสะอาดข้อมูล)
* การลบข้อมูลที่ซ้ำซ้อน (Duplicate Records) และแก้ไขข้อมูลที่พิมพ์ผิดพลาด (Inconsistent Data)
* **การเปรียบเทียบเชิงสถิติ:** ทดลองแทนที่ค่าว่างด้วย **Mean (ค่าเฉลี่ย)** และ **Median (ค่ามัธยฐาน)** เพื่อศึกษาผลกระทบต่อการกระจายตัวของข้อมูล

### ⚙️ Part 4: Feature Engineering (การเตรียมคุณลักษณะข้อมูล)
* แปลงข้อมูลประเภทหมวดหมู่ (Categorical Data) ให้เป็นตัวเลขเพื่อให้โมเดลประมวลผลได้
* ฝึกฝนการทำ **Label Encoding** และ **One-Hot Encoding** แยกตามความเหมาะสมของตัวแปร
